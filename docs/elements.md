# 游戏中元素

- `element`  
  游戏中**元素**
  - `block`  
    迷宫内**方格**的不同状态
    - `wall`  
      **墙点**, 阻挡人物移动
    - `road`  
      **路点**
  - `item`  
    迷宫内的**物品**
    - `dimd`  
      💎 _**diamond**_  
      **钻石**, 用于标记玩家是否能够过关
    - `coin`  
      🪙  
      **硬币**, 标记玩家本局分数
  - `ghost`  
    👻  
    **鬼怪**, 阻碍玩家游戏
  - `edpnt`  
    🚩 _**endpoint**_  
    **终点**
  - `fig`  
    😕 _**figure**_  
    玩家操控的**人物**
